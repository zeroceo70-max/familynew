# FamilyShare – Safety Through Trust and Consent

The world's first 100% consent-based family safety & locator app.  
No secret tracking. Ever.

## Live App
https://familynew.vercel.app (or your final domain)

## Features
- Real-time family circle with live Google Maps
- Consent-only location sharing (pause/stop anytime)
- Temporary check-in requests with approve/decline
- Public safety alerts board
- Supervised accounts for minors (QR enrollment + child-side acceptance)
- Email + phone verification
- 6-digit email confirmation code
- Full PWA – installable on iPhone & Android
- GDPR / CCPA / COPPA compliant
- One-click "Revoke All Access" and "Delete Account"
